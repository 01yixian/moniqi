# 韭菜模拟器 (Leek Simulator)

## 项目介绍
这是一个不仅扎心，而且扎肺的股市模拟器。
它完美复刻了散户在股市中的心路历程：
- **追涨杀跌**：看着涨，一买就跌。
- **抄底被套**：看着跌，一补更跌。
- **宏观忽悠**：各种高大上的数据，其实跟你的账户盈亏没有任何关系。

## 核心功能
1. **智能收割算法**：内置OU、GBM、Regime Switching等多种随机过程模型，但无论哪种模型，最终归宿都是归零。
2. **情绪价值**：提供"关灯吃面"、"天台排队"等沉浸式体验。
3. **极速亏损**：支持1倍到6倍速亏损，让你的资产缩水效率提升600%。

## 技术栈
- HTML5 Canvas (纯手搓K线)
- Vanilla JavaScript (无依赖，加载快)
- CSS3 (响应式布局，手机也能随时随地亏钱)

## 部署说明
直接将 `index.html` 部署到 GitHub Pages 或 Netlify 即可。
由于为了防止移动端网络请求超时，所有宏观数据（LPR、CPI等）均为"模拟生成"（即：瞎编的），请勿用于真实投资参考！
